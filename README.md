# Franager Backend

[![pipeline status](https://gitlab.com/franager/backend/badges/ci/pipeline.svg)](https://gitlab.com/franager/backend/commits/ci)

### installation
```
$ npm install
```

### run migrations
```
$ npm run migrate:dev
```

### seed fake data in dev DB
```
$ npm run seed
```

### run the project
```
$ npm run dev
```
